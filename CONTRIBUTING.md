## Creating an issue
Install [this script](../../raw/github/scripts/github_new_issue.user.js) to help you create issues OR use the following template:

> **Firefox:** {version}<br />
> **Greasemonkey:** {version}<br />
> **Script:** {name} {version}<br />
> **Url:** {url}<br />
> **Error Message:** {message}<br />
> **Obs.:** {observation}<br />
